# Workflow

## [N8N](https://n8n.io/)
An open-source, low-code workflow automation platform that enables technical teams to connect applications, services, and APIs through a visual drag-and-drop interface while retaining the flexibility to add custom JavaScript or Python code, deploy self-hosted for full data control, and build everything from simple integrations to complex AI-powered multi-step agents.


## [Dify](https://dify.ai/)
An open-source low-code platform for building and deploying AI-native applications, integrating essential tools like LLM management, RAG pipelines, and agentic workflows into a visual interface that enables rapid development without extensive coding.


## [Coze](https://www.coze.com/)
A no-code AI agent development platform, launched by ByteDance, that allows users to visually build, deploy, and manage intelligent chatbots with built-in plugins, knowledge bases, and multi-platform publishing.
