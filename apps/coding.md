# Coding

## CLI

### [Claude Code]()
An agentic AI coding assistant from Anthropic that runs in your terminal, autonomously executing complex engineering tasks—from planning and coding to testing and committing—by understanding your entire codebase and acting on natural language instructions. 

### [Codex]()
A cloud-based software engineering agent that autonomously writes, debugs, and tests code across multiple programming languages by executing tasks in parallel within isolated sandbox environments.

### [Gemini CLI](https://geminicli.com/)
An open-source AI agent from Google that brings the power of Gemini models directly to your terminal, enabling you to query codebases, automate tasks, and generate applications using natural language commands.


### [OpenCode]()
An open-source AI coding agent that runs in your terminal, allowing developers to write, debug, and refactor code using natural language while maintaining privacy and supporting 75+ LLM providers.

## Online

### [Replit](https://replit.com/)
A cloud-based integrated development environment (IDE) that lets you write, run, and deploy code directly from your browser, with built-in AI assistance and real-time collaboration features.

### [v0](http://v0.app/)
An AI-powered development platform by Vercel that generates full-stack, production-ready code from natural language prompts and integrates directly with existing GitHub repositories for seamless deployment.

### [Bolt.new](https://bolt.new/)
An AI-powered in-browser development environment that creates, runs, and deploys full-stack web applications from natural language prompts—with no local setup required, powered by WebContainers technology that runs Node.js directly in your browser.
